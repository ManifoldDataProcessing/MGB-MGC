# MGB-MGC: Multi-granularity clustering via modularity-based granular-balls

This is the official implementation for the paper "Multi-granularity clustering via modularity-based granular-balls", which is currently under review.

# Requirements

The code is developed with Python 3.11. Please install the following required packages:

- numpy (version 2.3.4)
- scipy (version 1.16.3)
- scikit-learn
- matplotlib(version 3.10.6)
- networkx
- python-igraph(version 1.0.0)
- leidenalg(version 0.11.0)

You can install them using pip, for example:

pip install numpy==2.3.4
pip install scipy==1.16.3

# run
python MGB-MGC.py
